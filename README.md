# Student Performance Analysis using Machine Learning

## Overview
This project analyzes and predicts student performance using machine learning and statistical techniques. It demonstrates how data-driven insights can be used to understand factors affecting academic performance.

---

## Features
- Data generation using NumPy
- Exploratory Data Analysis (EDA)
- Statistical analysis (mean, variance, distribution)
- Hypothesis testing using t-test
- Regression model to predict student scores
- Classification model to predict pass/fail outcomes
- Clustering to group students based on performance

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

---

## Results
- Strong correlation between study hours and performance
- Hypothesis testing confirmed statistical significance (p < 0.05)
- Regression model achieved high R\'b2 score
- Classification model achieved good accuracy
- Clustering revealed meaningful student groups

---

## Conclusion
Machine learning and statistical techniques effectively help in analyzing and predicting student performance. This project showcases the integration of data analysis, hypothesis testing, and ML models.
